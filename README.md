# Brahma Legal Pages

Legal documentation for the Brahma app by Happyloop Apps.

## Pages

- `index.html` - Home page
- `privacy.html` - Privacy Policy
- `terms.html` - Terms of Service
- `support.html` - Support & FAQ

## Deploying to GitHub Pages

1. Create a new GitHub repository (e.g., `brahma-legal`)
2. Push these files to the repo
3. Go to Settings > Pages
4. Set Source to "Deploy from a branch"
5. Select "main" branch and "/ (root)"
6. Save

Your site will be available at: `https://YOUR_USERNAME.github.io/brahma-legal/`

## URLs for Play Console

After deployment, use these URLs:

- **Privacy Policy:** `https://YOUR_USERNAME.github.io/brahma-legal/privacy.html`
- **Terms of Service:** `https://YOUR_USERNAME.github.io/brahma-legal/terms.html`
- **Support:** `https://YOUR_USERNAME.github.io/brahma-legal/support.html`

## Future Migration

When you get a custom domain:
1. Add CNAME file with your domain
2. Update GitHub Pages custom domain setting
3. Update URLs in Play Console and App Store Connect
